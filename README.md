# Coucou 🇫🇷

*Coucou* — a friendly French "hi!" — is a free, AI-era playbook for English speakers learning French: the smart method, the best 2026 AI tools, a copy-paste prompt cookbook, a study routine, and a downloadable tracker. Beginner → **DELF B2** (and French citizenship).

**Live site:** https://ektasengar.github.io/coucou/

## What's inside

- **The method** — AI as an amplifier, output-first, immersion, consistency.
- **Set your target** — CEFR levels, hours-to-B2 calculator, citizenship requirements (2026).
- **The learning stack** — six layers from structure to certification.
- **Smart AI tools (2026)** — Langua, Speak, ChatGPT Voice, Praktika, Claude — with the traps to avoid.
- **AI prompt cookbook** — 12 copy-paste prompts (tutor, role-plays, essay correction, flashcards, DELF oral sim…).
- **Routine, resources, cost & funding, certification, a worked example, and a 30-day quick-start.**
- **Downloadable tracker** — [`assets/French_Learning_Tracker.xlsx`](assets/French_Learning_Tracker.xlsx).

## Tech

Plain static HTML/CSS/vanilla JS — no build step. Interactivity (copy buttons, progress tracker, checklist, calculator, dark mode) is vanilla JS with `localStorage`. Hosted on GitHub Pages.

```
index.html        · the single-page site
css/style.css     · styles (light/dark)
js/main.js        · interactivity
assets/           · favicon + downloadable .xlsx tracker
404.html          · friendly not-found page
```

## Editing & redeploying

Edit the files and push to `main` — GitHub Pages redeploys automatically.

```bash
git add -A && git commit -m "Update playbook" && git push
```

## Notes

The tracker spreadsheet is generated by `build_tracker.py` (kept outside this repo). Prices, tools and immigration rules change over time — verify official requirements on [service-public.fr](https://www.service-public.fr) before relying on them.
